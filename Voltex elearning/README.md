# Voltex E-learning

Descripción corta del proyecto Voltex E-learning.

## Estructura
- src/: código fuente
- docs/: documentación
- assets/: imágenes y recursos

## Empezar (local)
1. Clona el repo:
   git clone git@github.com:MC-Edgar/Voltex-E-learning.git
2. Entra en la carpeta:
   cd Voltex-E-learning
3. Instala dependencias (si aplica) y ejecuta:
   <!-- reemplaza según tu stack -->
   npm install
   npm run dev
